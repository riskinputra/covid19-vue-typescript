# covid19-vue-typescript
Is project only for learn how to using VUE.js and Typescript and for monitoring CORONA Virus.

<p align="center">

## Install
```shell
yarn install
yarn start
```

## Features
- Equip with Vue.js, Vue-cli 3, Typescript, date-fns, lodash.
- Grid use Flexboxgrid
- Stylesheets in scss
